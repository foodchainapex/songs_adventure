# Songweirong Adventure

A fun HTML5 endless runner game.

## Play Online

https://yourusername.github.io/songs_adventure/

## How to Play

- Press **SPACE** or **Tap Screen** to jump
- Dodge obstacles, collect items
- Get the highest score!

## Run Locally

```bash
git clone https://github.com/yourusername/songs_adventure.git
cd songweirong-adventure
open index.html
```
